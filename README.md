# Moove CRM — Prospecção Maringá

App de CRM (1 arquivo, roda no navegador) para a prospecção dos SaaS da Moove em Maringá:
**Lava.AI, Sorria.AI, Realce.AI, Corta.AI**.

**Abrir:** https://marinasdearaujo.github.io/moove-crm/

Funcionalidades: pipeline (kanban), cadastro de contatos, templates de WhatsApp editáveis que abrem a conversa já preenchida, follow-ups (lead não esfria), analytics com metas. Os dados ficam salvos no próprio navegador (localStorage) de cada aparelho. Use **Backup/Importar JSON** para mover entre PC e tablet.
